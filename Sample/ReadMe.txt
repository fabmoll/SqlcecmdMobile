The folder contains a sample with a reference script (For more information: http://www.fabricemoll.com/post/SqlCmd-for-SQLCE-on-Windows-Mobile.aspx)

The shortcut to execute is "apply-update.lnk".

Don't forget to edit the shortcut to change the database name or the execution path.

You need to change the content of AddFirstUser.sqlce and AddSecondUser.sqlce to match your table structure!
